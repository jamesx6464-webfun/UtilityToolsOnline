# 🧰 UtilityTools Online

A simple and fast collection of everyday utility tools built with **HTML, CSS, and JavaScript**.  
UtilityTools Online provides lightweight tools for calculations, conversions, and basic utilities — all running directly in the browser.

---

## 🚀 Live Demo

https://jamesx6464-webfun.github.io/UtilityToolsOnline/

---

## ✨ Features

- 🧮 Built-in calculator
- 🌡️ Temperature converter (Celsius / Fahrenheit)
- 💱 Currency converter
- ⚖️ BMI calculator
- 📏 Unit converter
- 🌙 Dark mode toggle
- 📱 Responsive design for mobile and desktop
- ⚡ Fast client-side performance (no backend)

---

## 🧭 Navigation

The site includes multiple pages:

- Home

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages (hosting)

---
## 🌙 Dark Mode

The website includes a built-in dark mode toggle 🌙  
Users can switch themes using the button in the navigation bar.

Theme preference is handled in JavaScript (no external libraries required).

---

## ⚙️ How It Works

- Each tool is a separate HTML page
- JavaScript handles calculations and conversions
- No backend or database is used
- Everything runs locally in the browser

This makes the site:
- fast ⚡
- lightweight 🪶
- privacy-friendly 🔒

---

## 📱 Mobile Friendly

UtilityTools Online is fully responsive:
- Works on mobile phones
- Works on tablets
- Works on desktop screens
- Navigation adapts to smaller screens
